A comprehensive web application for exploring and discovering information about countries around the world.

## What This Application Does

Country Explorer is designed to help users learn about different countries, compare their characteristics, and track weather conditions in their capital cities. Whether you're planning a trip, conducting research, or simply curious about the world, this application provides an intuitive way to explore global data.

## Key Features

### Browse and Search Countries

Explore a comprehensive database of countries from around the world. Use the search functionality to quickly find specific countries by name, or browse through the complete list to discover new places.

### Advanced Filtering

Narrow down your search using multiple filter options:

- Filter by continent or region to focus on specific geographic areas
- Set population ranges to find countries of similar size
- Filter by life expectancy to compare quality of life indicators

### Sort and Organize

Sort countries by various criteria to find exactly what you're looking for:

- Alphabetically by name
- By population size
- By life expectancy rates

### Detailed Country Information

View comprehensive details about each country including:

- Population statistics
- Geographic information including capital city
- Life expectancy data
- Neighboring countries (borders)
- Time zones
- Official languages
- Government system

### Weather Information

Check real-time weather conditions in any country's capital city. The weather data includes current temperature, conditions, and other meteorological information to help you understand what it's like there right now.

### Compare Countries

Place two or more countries side by side to compare their characteristics. This feature is particularly useful for:

- Educational purposes
- Travel planning
- Research projects
- General curiosity about how different nations compare

### Save Favorites

Create a personalized collection of countries you're interested in. Mark countries as favorites to quickly access them later without searching through the entire database.

### Export Data

Download country information in multiple formats for offline use or further analysis:

- Export as CSV (spreadsheet format) for data analysis
- Export as PDF for presentations or printing

### Statistics Dashboard

View aggregated statistics and insights about the countries in the database, helping you understand global patterns and trends.

## User Account Features

The application includes secure user authentication, allowing you to:

- Create and manage your personal account
- Save your favorite countries across sessions
- Access your personalized dashboard
- Update your profile information
- Change your password
- Enable two-factor authentication for enhanced security

## Getting Started

Once you create an account and log in, you'll be taken to your dashboard where you can immediately start exploring countries. Use the search bar to find specific countries, or apply filters to discover countries that match your interests.

Click on any country to view its detailed information page, where you'll find comprehensive data and current weather conditions for its capital city. Use the compare feature to analyze multiple countries at once, and don't forget to mark your favorites for easy access later.

## Data Sources

The application combines information from multiple reliable sources to provide accurate and up-to-date information about countries worldwide, including geographic data, demographic statistics, and real-time weather information.

### APIs Used

The application integrates with the following external APIs to deliver comprehensive country information:

**REST Countries API**
- Provides detailed country information including borders, time zones, and geographic data
- Used to enrich country profiles with additional metadata
- Delivers information about neighboring countries and regional relationships

**Open-Meteo Weather API**
- Supplies real-time weather data for capital cities worldwide
- Provides current temperature, humidity, wind speed, and weather conditions
- Offers multi-day forecasts for planning purposes
- Free and open-source weather service with no API key required
